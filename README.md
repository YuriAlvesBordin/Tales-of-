# Tales of Arroba

**Tales of Arroba** is a **procedural open-world** game made with **ASCI** where survival is your greatest challenge!

### Controls:
- **Move:** `W`, `A`, `S`, `D`
- **Shoot:** `Arrow keys`

### Objective:
- Survive as long as possible by **eliminating enemies** that appear during your journey.
